Custom Component: Progress Indicator Gauge

A way of building a custom component in plain JavaScript with no dependencies and no Web Components and no Custom Elements. This type of custom component is plain HTML/JS and can be thus be used inside any framework. It is never subject to framework or version change. It never goes out of fashion. It needs no tools for building. This is not a library. It is just way of writing code.
Note that style and core property values are configured via CSS.
